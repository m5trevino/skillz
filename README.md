# skillz

Live skills imported via [skillz.save-aichats.com](https://skillz.save-aichats.com).

Each skill lands here as a single `<slug>.md` file with YAML frontmatter
(`name`, `description`, `tags`, `source`) plus the original markdown body.

Install any skill from this repo:

```
npx skills add m5trevino/skillz
```
